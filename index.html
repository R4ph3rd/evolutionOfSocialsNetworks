Evolution of features of social networks

Les points de couleurs représente les sorties de features sur la base de leur temporalité et le réseaux auquelle elle appartiennent, les différentes couleurs différencie les catégories des features nous catégorison la temporalité par des aires pour les réseaux sociaux,survoler la feature pour en savoir plus et cliquer sur une feature pour découvrir les liens entre les features similaires .
